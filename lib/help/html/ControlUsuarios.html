Ventana del Control de Usuarios

Esta es la ayuda de la ventana Control de Usuarios.

La ventana de Control de Usuarios sirve para controlar en el almacenamiento todos los datos de los usuarios. Cuenta con una tabla la cual se puede utilizar dandole a los botones con la celda que eligas para insertar borrar o modificar cualquiera, para mostrar los datos de un usuario es necesario girar la rueda del raton con el puntero encima del usuario que se quiere mostrar.
